# Radius Helper

Authors: Tuma, Tymamatyty

## EN
Radius Helper is a HolyWorld-focused territory utility.

- Point 1 + Point 2 selection.
- Shows where to place private markers to fence the area.
- Shows where to place TNT for territory cleanup.

This is a practical utility mod built for a specific server workflow.

## RU
Radius Helper - утилита для территории под HolyWorld.

- Выбор точки 1 и точки 2.
- Подсказывает, где ставить приваты для ограждения.
- Подсказывает, где ставить TNT для очистки территории.

Мод практический и заточен под конкретный серверный сценарий.
